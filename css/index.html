# 回流和重绘

  • 回流reflow 指的是页面元素布局 结构等改变 浏览器会重新渲染 包括窗口大小改变，激活css伪类 例如：hover 调用clientWidth offsetWidth 等方法
  • 重绘repaint 指的是页面元素只改变了color background-color visibility 等css 浏览器会重新绘制
  • 因此回流>重绘 发生回流就一定会重绘 反之不一定
  • 尽量避免页面的回流 因此经常改变的dom可以脱离文档流 先display:none 后面显示
