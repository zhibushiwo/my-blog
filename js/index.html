# this

this对象的一大原则就是谁调用指向谁 没有人调用的话只想window 对于new 后的this 就绑定这个new出来的实例

# call apply bind 的相同与不同

  • 三者的主要目的就是改变this的指向
  • fn.call(obj,arg1,arg2,arg3...) 指的是obj来调用fn的这个方法，后面是参数，可以简写成...arg arg是一个数组
  • fn.apply(obj,[arg1,arg2,arg3...]) 同call，参数的形式不一样，这里直接传的是参数的数组
  • const fn2 = fn.bind(obj) ; fn2() 和call和apply相比 bind不能自执行 只能返回一个新的函数 再去调用 而且也不能添加参数

# 网络

# get post的区别

  • get传递是通过url 长度有限制 回退没有影响 可以缓存 会保留记录
  • post通过HTTP 里面 长度没有限制 回退重新提交 不可以缓存 不会保留记录

# http 响应码

  • 1XX 临时响应
  • 2XX 请求成功 200 成功
  • 3XX 重定向
  • 4XX 请求错误 400请求错误 401未通过授权 403不允许访问 404请求未找到
  • 5XX 服务器错误 500 服务器内部错误 503 服务不可用 504超时了
